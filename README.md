# Java Socket Chat Application

This project is a basic client-server chat application built using Java Sockets and Java Swing. It demonstrates how TCP socket communication works by allowing a client to connect to a server and exchange messages in real time.

The server listens for incoming client connections, receives messages, and echoes them back to the client. The client provides a simple graphical user interface (GUI) built with Swing, allowing users to send and receive messages seamlessly. A background thread is used to listen for incoming messages so that the interface remains responsive.

## Features

* Client-server communication using TCP sockets
* Java Swing graphical user interface
* Real-time message exchange
* Multithreaded client for continuous message listening
* Simple and beginner-friendly implementation for learning Java networking
